# Mattress Tracker

A custom Home Assistant integration to track when you last flipped and rotated your mattress.

## Features
- Track flipping between custom-named sides (e.g., Firm/Soft).
- Track 180-degree rotation (Top at Head / Top at Foot).
- Log actions for "today" or a specific historical date.
- Persistent state across Home Assistant restarts.

## Installation
Add this repository to HACS as a custom repository.

## Configuration
Go to Settings -> Devices & Services -> Add Integration and search for "Mattress Tracker".
